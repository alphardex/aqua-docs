## 可定制变量

### .breadcrumb

| 名称                | 作用               |
| ------------------- | ------------------ |
| --divider           | 分隔符             |
| --divider-color     | 分隔符的颜色       |
| --link-color        | 导航链接的颜色     |
| --active-link-color | 当前导航链接的颜色 |
