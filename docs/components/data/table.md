## 可定制变量

### .table

| 名称                     | 作用                   |
| ------------------------ | ---------------------- |
| --table-row-border-color | table 行边框的颜色     |
| --table-thead-color      | table 头文字的颜色     |
| --table-thead-bg         | table 头背景的颜色     |
| --table-td-padding       | table 单元格的内边距   |
| --table-td-align         | table 单元格的文本对齐 |
