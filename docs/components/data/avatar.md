## 可定制变量

### .avatar

| 名称            | 作用              |
| --------------- | ----------------- |
| --avatar-width  | avatar 的长度     |
| --avatar-bg     | avatar 的背景     |
| --avatar-border | avatar 的边框     |
| --avatar-radius | avatar 的圆角长度 |
