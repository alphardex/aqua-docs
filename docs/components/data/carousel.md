## 可定制变量

### .carousel

| 名称                   | 作用                      |
| ---------------------- | ------------------------- |
| --carousel-width       | carousel 的宽度           |
| --carousel-height      | carousel 的高度           |
| --carousel-link-width  | carousel 导航点的长度     |
| --carousel-link-margin | carousel 导航点之间的间距 |
| --carousel-link-color  | carousel 导航点的颜色     |
