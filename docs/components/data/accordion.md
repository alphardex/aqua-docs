## 可定制变量

### .accordion

| 名称           | 作用                            |
| -------------- | ------------------------------- |
| --text-color   | 文本的颜色                      |
| --active-color | accordion-item 展开时文本的颜色 |
| --line-color   | 下划线的颜色                    |
