## 可定制变量

### .input-spinner

| 名称                    | 作用                           |
| ----------------------- | ------------------------------ |
| --spinner-default-color | 默认情况下计数器文本的颜色     |
| --spinner-active-color  | 鼠标悬浮情况下计数器文本的颜色 |
| --spinner-max-width     | 计数器中间部分的最大宽度       |
