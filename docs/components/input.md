## 可定制变量

### .form-input-material

| 名称                  | 作用                    |
| --------------------- | ----------------------- |
| --default-color       | 默认情况下label的颜色   |
| --invalid-color       | 校验不通过时label的颜色 |
| --valid-color         | 校验通过时label的颜色   |
| --border-bottom-color | 底部边框的颜色          |
