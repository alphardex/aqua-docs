## 可定制变量

### .close-icon

| 名称                     | 作用                   |
| ------------------------ | ---------------------- |
| --close-icon-color       | 关闭图标的背景的颜色   |
| --close-icon-cross-color | 关闭图标中间叉叉的颜色 |

### .chevron-icon

| 名称                 | 作用               |
| -------------------- | ------------------ |
| --chevron-rotate-deg | 箭头图标的旋转角度 |
