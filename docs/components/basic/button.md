## 可定制变量

### .btn

| 名称        | 作用          |
| ----------- | ------------- |
| --btn-color | button 的颜色 |

### .btn-group

| 名称             | 作用         |
| ---------------- | ------------ |
| --divider-color  | 分割线的颜色 |
| --divider-length | 分割线的长度 |
