## 可定制变量

### .divider

| 名称                   | 作用                   |
| ---------------------- | ---------------------- |
| --divider-text         | divider 的内容         |
| --divider-border-color | divider 的颜色         |
| --divider-color        | divider 文字的颜色     |
| --divider-text-align   | divider 文字的对齐方式 |
