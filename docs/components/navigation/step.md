## 可定制变量

### .step

| 名称                 | 作用                  |
| -------------------- | --------------------- |
| --step-circle-color  | step 步骤圈边框的颜色 |
| --step-circle-bg     | step 步骤圈背景的颜色 |
| --step-line-color    | step 步骤线的颜色     |
| --step-default-color | step 默认状态下的颜色 |
| --step-active-color  | step 激活状态下的颜色 |
