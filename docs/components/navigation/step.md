## 可定制变量

### .step

| 名称              | 作用             |
| ----------------- | ---------------- |
| --circle-color    | 步骤圈边框的颜色 |
| --circle-bg-color | 步骤圈背景的颜色 |
| --line-color      | 步骤线的颜色     |
| --default-color   | 默认状态下的颜色 |
| --active-color    | 激活状态下的颜色 |
