## 可定制变量

### .form-switch

| 名称                | 作用                     |
| ------------------- | ------------------------ |
| --switch-color      | switch 背景的颜色        |
| --ball-color        | switch 珠子的颜色        |
| --ball-active-color | 打开 switch 时珠子的颜色 |
| --line-color        | switch 中间线的颜色      |
